Pemrograman Website Dasar

Kata Pengantar HTML CSS JavaScript

Kata Pengantar

Selamat datang di panduan Pemrograman Website Dasar. Panduan ini dirancang agar pemula memahami pembuatan website dari awal. Kamu akan belajar HTML, CSS, dan JavaScript step-by-step dengan contoh kode nyata.

Gunakan tombol navigasi di atas untuk berpindah antar section. Setiap sub-section memiliki tombol pop-up tips untuk membantu belajar lebih efektif.

HTML

Struktur Dasar

HTML membentuk struktur halaman. Contoh dasar:

<!DOCTYPE html>
<html>
<head>
  <title>Contoh</title>
</head>
<body>
  <h1>Halo Dunia</h1>
  <p>Paragraf pertama</p>
</body>
</html>

Heading & Paragraf

HTML memiliki tag heading H1-H6 dan paragraf P:

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<p>Ini paragraf teks.</p>

Link, Daftar, dan Gambar

Membuat tautan, daftar, dan menambahkan gambar:

<a href='https://example.com'>Kunjungi Example</a>
<ul><li>Item 1</li><li>Item 2</li></ul>
<img src='gambar.jpg' alt='Contoh Gambar' width='200'/>

Tabel

Membuat tabel sederhana:

<table border="1">
  <tr><th>Nama</th><th>Umur</th></tr>
  <tr><td>Budi</td><td>20</td></tr>
</table>

Form & Input

Contoh form HTML:

<form>
  <input type='text' placeholder='Nama'>
  <input type='submit' value='Kirim'>
</form>

CSS

Warna, Font & Teks

Mengatur tampilan teks:

body {background:#f0f0f0;color:#333;font-family:Arial;}
h1 {color:#00f;text-align:center;}
p {color:#555;font-size:16px;}

Margin, Padding & Border

div {margin:10px;padding:15px;border:1px solid #000;}

Layout & Flexbox

.container {display:flex;justify-content:space-around;}
.item {width:100px;height:100px;background:#0ff;}

Animasi & Transition

button:hover {background-color:#0ff;transform:scale(1.1);transition:all 0.3s;}

JavaScript

Variabel & Tipe Data

Mendeklarasikan variabel:

let nama = 'Budi';
const umur = 20;
var aktif = true;

Operator & Kondisi

if(umur >= 18){
  alert('Dewasa');
}else{
  alert('Belum dewasa');
}

Looping

for(let i=1;i<=5;i++){
  console.log(i);
}

Fungsi & Event

function sapa(nama){
  alert('Halo '+nama);
}
document.getElementById('btn').addEventListener('click', function(){
  sapa('Budi');
});

DOM & Manipulasi Konten

document.getElementById('demo').innerHTML = 'Teks baru';

Validasi Form

document.getElementById('form').addEventListener('submit', function(e){
  e.preventDefault();
  const nama = document.getElementById('nama').value;
  if(nama==='') alert('Nama harus diisi!');
});
