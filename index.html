Pemrograman Website Dasar

Belajar langkah awal membuat website

1. Apa itu Website?

Website adalah sekumpulan halaman yang saling terhubung dan dapat diakses melalui internet menggunakan browser.

2. Struktur Dasar HTML

Setiap website biasanya dibuat menggunakan HTML, CSS, dan JavaScript. Berikut struktur HTML paling dasar:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Website</title>
  </head>
  <body>
    <h1>Halo, dunia!</h1>
  </body>
</html>

3. CSS untuk Tampilan

CSS digunakan untuk mengatur tampilan, seperti warna, font, dan layout.

4. JavaScript untuk Interaksi

JavaScript membuat website lebih interaktif, seperti tombol, animasi, dan validasi form.
