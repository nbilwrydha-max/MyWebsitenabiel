Pemrograman Website Dasar

Pengenalan HTML CSS JavaScript

Pengenalan

Pemrograman website dasar adalah fondasi untuk membuat halaman web yang dapat diakses melalui browser. Dengan memahami dasar HTML, CSS, dan JavaScript, kamu dapat membuat website interaktif, menarik, dan responsif untuk semua perangkat, termasuk HP dan tablet.

HTML (HyperText Markup Language)

HTML adalah bahasa markup untuk membuat struktur halaman web. Elemen dasar HTML termasuk <h1> sampai <h6> untuk judul, <p> untuk paragraf, <a> untuk tautan, dan <img> untuk gambar.

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan website. Dengan CSS, kamu dapat menentukan warna, ukuran font, layout, animasi, dan efek visual lainnya. Misalnya, kamu bisa menambahkan efek hover pada tombol agar berubah warna ketika disentuh atau diklik.

JavaScript

JavaScript memungkinkan website menjadi interaktif. Dengan JavaScript, kamu bisa menambahkan logika, menangani klik tombol, validasi form, animasi, dan banyak lagi. JavaScript bekerja bersama HTML dan CSS untuk menciptakan pengalaman pengguna yang dinamis.
