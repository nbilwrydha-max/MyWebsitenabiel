Kata Pengantar

Selamat datang di panduan Pemrograman Website Dasar. Buku ini dibuat khusus bagi pemula yang ingin memahami cara membuat website dari awal. Kamu akan belajar mulai dari struktur halaman, penataan visual, hingga membuat interaksi menggunakan JavaScript.

Setiap section dirancang panjang dan detail agar seperti ebook atau majalah digital. Kamu bisa membaca di HP maupun PC, dan semua contoh bisa langsung dicoba sendiri.

Gunakan tombol navigasi di atas untuk berpindah antar section, dan tombol Light / Dark Mode untuk menyesuaikan tampilan mata. Di setiap section ada tombol pop-up untuk tips tambahan agar belajar lebih efektif.

Pembahasan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membangun struktur website. Dengan HTML, kamu bisa membuat judul, paragraf, tautan, gambar, tabel, daftar, dan banyak elemen lain yang membentuk halaman web.

Contoh dasar HTML:

<!DOCTYPE html>
<html>
<head>
  <title>Contoh HTML</title>
</head>
<body>
  <h1>Halo Dunia</h1>
  <p>Ini paragraf pertama</p>
</body>
</html>

Tips penting: selalu menutup tag HTML dengan benar dan menulis DOCTYPE di awal agar browser menampilkan halaman dengan baik.

Pembahasan CSS

CSS (Cascading Style Sheets) digunakan untuk mempercantik tampilan website. Kamu dapat mengatur warna, font, layout, jarak antar elemen, dan membuat animasi sederhana.

Contoh CSS dasar:

body {
  background-color: #f0f0f0;
  font-family: Arial, sans-serif;
  color: #333;
}
h1 {
  color: #00f;
  text-align: center;
}

Tips: Gunakan class dan ID untuk mengatur style elemen tertentu, dan manfaatkan Flexbox atau Grid untuk membuat layout responsif.

Pembahasan JavaScript

JavaScript membuat website menjadi interaktif. Kamu dapat menangani klik tombol, validasi form, mengubah konten secara dinamis, dan membuat animasi.

Contoh JavaScript sederhana:

<button onclick="alert('Halo!')">Klik Aku</button>

Tips: Mulailah dengan memahami DOM (Document Object Model), event, dan fungsi sebelum mempelajari framework seperti React atau Vue.
