Pemrograman Website Dasar

Kata Pengantar HTML CSS JavaScript

Kata Pengantar

Selamat datang di panduan Pemrograman Website Dasar. Panduan ini dirancang agar pemula memahami pembuatan website dari awal melalui contoh nyata dan tips praktis. Kamu akan belajar HTML, CSS, dan JavaScript step-by-step.

Gunakan tombol navigasi untuk berpindah antar section. Setiap sub-section memiliki tombol pop-up untuk tips tambahan agar belajar lebih efektif.

HTML

Struktur Dasar

HTML membentuk struktur halaman. Contoh dasar:

<!DOCTYPE html>
<html>
<head>
  <title>Contoh</title>
</head>
<body>
  <h1>Halo Dunia</h1>
  <p>Paragraf pertama</p>
</body>
</html>

Teks & Gambar

Menambahkan teks, gambar, dan atribut alt:

<p>Ini paragraf contoh.</p>
<img src='gambar.jpg' alt='Contoh Gambar' />

Link & List

Membuat tautan dan daftar:

<a href='https://example.com'>Kunjungi Example</a>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

CSS

Warna & Font

CSS mempercantik tampilan:

body {background:#f0f0f0;color:#333;font-family:Arial;}
h1 {color:#00f;text-align:center;}

Layout & Flexbox

Mengatur layout:

.container {display:flex;justify-content:space-around;}

Animasi & Hover

Efek hover pada tombol:

button:hover {background-color:#0ff;transform:scale(1.1);}

JavaScript

DOM & Event

Membuat interaksi tombol:

<button id="btn">Klik Aku</button>
<script>
document.getElementById('btn').addEventListener('click', function(){
  alert('Halo Dunia!');
});
</script>

Validasi Form

Contoh validasi sederhana:

<form id="form">
  <input type="text" id="nama" placeholder="Nama">
  <button type="submit">Kirim</button>
</form>
<script>
document.getElementById('form').addEventListener('submit', function(e){
  e.preventDefault();
  const nama = document.getElementById('nama').value;
  if(nama==='') alert('Nama harus diisi!');
  else alert('Halo, '+nama);
});
</script>
