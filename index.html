Apa itu Website?

Website adalah kumpulan halaman yang saling terhubung dan dapat diakses melalui internet menggunakan browser. Website biasanya ditulis dengan HTML untuk struktur, CSS untuk tampilan, dan JavaScript untuk interaktivitas.

Contoh jenis website: blog, toko online, portofolio, media sosial, hingga aplikasi berbasis web.

Langkah Membuat Website Dasar

1. Buat file index.html sebagai halaman utama.
2. Tambahkan CSS untuk mempercantik tampilan.
3. Gunakan JavaScript untuk menambah interaksi dinamis.
4. Simpan file di hosting gratis seperti GitHub Pages.
5. Bagikan link websitemu agar bisa diakses orang lain.
